# Digital Tools for Ocean Conservation

This repository contains the code for a web-based showcase demonstrating how modern digital technologies can bridge complex scientific research and public outreach. The page highlights the use of Augmented Reality (AR) and sophisticated data visualisation to enhance the clarity, accessibility, and impact of scientific findings, with a focus on marine movement ecology.

### [**View the live project here**](https://miguelgandra.github.io/digital-scicomm/)

![A screenshot of the hero section of the webpage.](https://miguelgandra.github.io/digital-scicomm/media/page-screenshot.png)

##

About This Project

This project serves as a digital exhibit to show how raw, complex scientific data can be transformed into engaging, interactive experiences. Grounded in rigorous movement ecology research, this work aims to make science more interpretable and memorable for diverse audiences.
